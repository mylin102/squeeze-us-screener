# Squeeze Stock Screener (Taiwan Market) v1.0

Automated stock screener for the Taiwan market using Squeeze Momentum logic and advanced pattern recognition.

## Key Features
- **High-Performance Scanning**: Hybrid multi-threaded/multiprocessing engine.
- **Advanced Patterns**: TTM Squeeze, Houyi Shooting the Sun, and Whale Trading (Multi-timeframe).
- **Fundamental Filtering**: Filter by Market Cap, Volume, and a percentile-based **Value Score**.
- **Professional Reporting**: Export results to CSV, JSON, and Markdown summaries.
- **Automated Visualization**: Generates professional candlestick charts with indicator overlays.
- **Daily Automation**: Pre-configured GitHub Actions for daily scans and LINE Bot notifications.

## Quickstart

### Installation
```bash
pip install .
```

### Run a Scan
```bash
# Scan for top value stocks currently in a Squeeze
squeeze scan --min-score 0.8 --plot --export
```
